# Pneumatic Gripper for ABB IRB 120

Two-finger pneumatic gripper designed for an anthropomorphic industrial robot (ABB IRB 120).
The gripper is intended for handling hollow steel cones (S235) with variable dimensions.

The project covers kinematic structure selection, displacement and force analysis using the
vector polygon method, gripping force calculations including inertia forces, and FEA stress
verification. The actuator is a single-acting pneumatic cylinder.
